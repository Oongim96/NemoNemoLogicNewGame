## 잉크 · 베가

**「베가, 원색의 화가」** · `char_vega` · SSR 한정

| | |
| --- | --- |
| 컨셉 | `잉크` |
| 스피드 핵심 | **칸 맞출 때마다 번짐** — 손으로 다 안 채워도 됨 |

### 퍼즐 패시브 — 「원색 번짐」

- 칸 **정답** 시: 십자 **2칸** 상태 공개 (확정 채색 X 아님, **힌트만**)
- **줄/열 완성** 시: 맞닿은 줄 숫자 **1줄 자동 해석** (구역당 최대 3회)
- 덱 `잉크` 3장+: 번짐 범위 **십자 → 3×3 외곽 1칸**

### 전투 패시브 — 「잉크 지배」

- 전투 시작 `잉크` 스택 **+2**
- `잉크` 카드 쿨 **-1** (파티 전체, **최소 1**)
- **턴 잉크 3장+** → 범위 스플래시 **75%** (기본 60% 상향)

### ult — 「캔버스 붓질」

- 구역 **1회**: 빈 칸 중 **확정 정답 5칸** 랜덤 공개 (대형 퍼즐 탈출)

### 고유 · 전용 카드

| card_id | 이름 | 등급 | 요약 |
| --- | --- | --- | --- |
| char_vega_001 | **원색 붓** | 에픽 | 칸 정답=잉크+2. 전투: 스택 소모 없이 범위 1회 |
| ink_006 | 잉크 해일 | 희귀 | (공용) 폭발 보조 |

### 덱 시너지

| 카드 | 시너지 |
| --- | --- |
| 연쇄 채색 | 번짐 + 연쇄 X → **줄 단위 빠른 마감** |
| 붉은 잉크 | 실수 스택 대신 **번짐으로 줄 완성 가속** |
| 원색 폭발 | 베가 패시브로 **턴 5 잉크 폭발** 도달 빠름 |
| 잉크 격자 | 줄 완성 시 잉크 범위 **초반부터** |

**추천 덱:** `잉크` 위주 6장+ → 범위·폭발 뽕맛

## 불꽃 · 이그니스

**「이그니스, 재의 사냥꾼」** · `char_ignis` · SSR 한정

| | |
| --- | --- |
| 컨셉 | `불꽃` |
| 스피드 핵심 | **콤보 → 줄 폭발** |

### 퍼즐 패시브 — 「연소 연쇄」

- **연속 정답 3**: 해당 줄/열 **숫자 전부** 1초 강조 + 오류 숫자 1개 표시
- **연속 5**: 그 줄/열 **빈 칸 1개** 확정 정답 공개
- 콤보 끊김 없이 구역 클리어 시 — 다음 구역 시작 **화상 스택 UI** (전투 +1 화상)

### 전투 패시브 — 「재의 연타」

- `불꽃` 공격 카드 = **연타 +0.5** (2장이면 3연타로 계산, 반올림)
- **턴 연타 3+** 화상 **2스택** (기본 1)
- **턴 연타 5+** 불길 통로 **항상** 발동 (마지막 ×2)

### ult — 「소각선」

- **한 줄 + 한 열** 선택 → 교차 칸 제외하고 해당 줄·열 **확정 오답(X) 일괄 처리** (정답 칸은 유지)

| flame_005 | 재의 선포 | 에픽 | (공용) |

### 덱 시너지

| 카드 | 시너지 |
| --- | --- |
| 불씨 튀기 | 콤보 유지하며 **연타 쌓기** |
| 소각 | 이그니스로 **화상 3 스택** 진입 쉬움 |

| 점화 | 콤보 스택 = 퍼즐·전투 동시 성장 |

**추천 덱:** `불꽃` 공격 CD1 다수 → 연타·화상

## 달빛 · 디아나

**「디아나, 차가운 관측자」** · `char_diana` · SSR 한정

| | |
| --- | --- |
| 컨셉 | `달빛` `힌트` |
| 스피드 핵심 | **약점 줄 통째로 읽기** |

### 퍼즐 패시브 — 「월하 관측」

- 구역 시작: **약점 후보 줄 1개** 전체 숫자 해석 완료 표시
- 줄 완성 시: 그 줄에 **약점 칸** 있으면 **확정 표시**
- 구역당 **실수 1회** → 오답이 아닌 **「의심」** 처리 (HP·저주 없음)

### 전투 패시브 — 「월식 지배」

- `달빛` 크리 확률 **+25%** (파티)
- **턴 달빛 2장+** → 월식 약점 (기본 3장에서 **완화**)
- **턴 달빛+힌트** 동시 발동 → 약점 피해 **×2.5** (기본 2.0)

### ult — 「만월 스캔」

- **3×3** 영역 내 **모든 정답 칸** 3초간 표시 (채우지는 않음)

| 스피드 핵심 | **실수 무시하고 밀기** |

### 퍼즐 패시브 — 「요새 돌진」

- 구역당 **실수 2회** HP 피해 **0** (3회째부터 정상)

- 이번 구역 **실수 전부** 되돌리기 (칸 상태·HP) — **1회**

| --- | --- |
| 컨셉 | `격자` |

### 퍼즐 패시브 — 「완성 연쇄」

- **줄/열 완성** 시: 인접 줄 숫자 **자동 검증** + **골드 +3**
- 런 내 줄 완성 수가 전투 `line_count_scaling`에 **×1.5** 반영

### 전투 패시브 — 「거대 설계」

- **덱 격자 4+** → 거대 격자 (기본 5에서 **완화**)
