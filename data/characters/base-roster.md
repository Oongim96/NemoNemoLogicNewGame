# 기본 로스터 (SR)

[← 캐릭터 README](./README.md) · [한정 SSR](./limited-characters.md)

스토리·기본 가챠. **한정 SSR보다 약하지만** 컨셉 입문용.

---

## 루나 `char_luna` · SR

| | |
| --- | --- |
| 컨셉 | `달빛` `힌트` |
| 역할 | 정밀·힌트 |

**퍼즐 패시브** — 선택한 **가로 또는 세로 줄 1개** 숫자 자동 분석 (구역당 2회)

**전투 패시브** — `달빛` 태그 카드 발동 시 **30%** 같은 턴 재발동

**ult** — 3×3 영역 정답 표시 (구역 1회)

**고유 카드** — moon_004 월식의 눈, moon_005 만월 기도

## 미오 `char_mio` · SR

| | |
| --- | --- |

| 역할 | 드래프트·보물 |

**퍼즐 패시브** — 드래프트 **1회 재굴림**/런 (무료)

## 세라핀 `char_sera` · SR

| | |
| --- | --- |
| 컨셉 | `잉크` |
| 역할 | 힐·잉크 연계 |

## SR vs SSR

| | SR (이 4인) | SSR 한정 |
| --- | --- | --- |
| 퍼즐 | 보조·완화 | **속공·대량 해석** |
| 전투 | +15~20% | **임계 완화·배율 대폭** |
| 고유 카드 | 희귀 1~2장 | **에픽급 전용** 1장 포함 |
