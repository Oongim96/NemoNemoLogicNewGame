# 05. 구역·보상

[← 난이도 확장](./04-difficulty.md) · [목차](./README.md) · [다음: 잉크 덱 빌드 →](./06-ink-deck-build.md)

**한 구역 = 네모네모 로직 퍼즐 1개.**  
구역 퍼즐을 **전부 완료했을 때** 보상이 **1회** 열린다.

## 구역 보상 흐름

```txt
[런 시작] 구역마다 보상 미리 배정 → 맵 타일 중앙에 아이콘 표시
  ↓
구역 선택 → 네모네모 퍼즐 풀기
  → 구역 100% 완료!
  → 미리 정해진 보상 1회 지급
  → 큰 그림 해당 구역 밝혀짐 → 맵으로
```

**능력 3택1**이 가장 자주 나온다. 상세는 [잉크 덱 빌드](./06-ink-deck-build.md).

## 구역 클리어 보상 (맵 3종 → 실제 지급)

| 맵 표시 | 배정 확률 | 클리어 시 |
| --- | --- | --- |
| **🃏 3택1** | ~72% | 카드 3장 중 1장 + 골드 10~18 |
| **🏪 상점** | ~10% | 상점 UI |
| **✨ 이벤트** | ~18% | 아래 중 랜덤 |

**이벤트 구역 세부** (클리어 때 롤):

| 세부 | 비중 |
| --- | --- |
| 골드 | 6 |
| 이벤트 | 5 |
| 회복 | 4 |
| 함정 | 3 |

## 맵 표시 (구역 선택)

런 시작 시 각 구역이 **3종**으로 미리 배정되고, 맵 타일 **가운데**에 표시된다.

| 아이콘 | 맵 표시 | 클리어 시 |
| --- | --- | --- |
| 🃏 | 3택1 | 카드 3택1 + 골드 보너스 |
| 🏪 | 상점 | 상점 UI |
| ✨ | 이벤트 | 골드 / 회복 / 함정 / 이벤트 중 **랜덤** |

골드·회복·함정·특수 이벤트는 맵에서는 모두 **✨ 이벤트**로 보이고, 퍼즐을 깨야 실제 내용이 정해진다.

**2×2 이상** 맵(구역 4개+)에서는:

- **상점(🏪) 최소 1구역** 반드시 포함
- **3택1(🃏) 최소 2구역** 반드시 포함 (부족하면 ✨ 이벤트 구역을 3택1으로 치환)

## 구역 완료 시 항상

- 큰 그림 해당 구역 **밝혀짐**
- 클리어 보너스 골드 +10
- 위에서 **미리 배정된 보상** 1종 지급

## 구역 타입 (보상 성향)

| 타입 | 특징 |
| --- | --- |
| 보물 구역 | 3택1 등급 UP |
| 상점 구역 | 상점·골드 비중 UP |
| 이벤트 구역 | 이벤트 비중 UP |
| 함정 구역 | 함정·저주 비중 UP |
| 혼합 구역 | 기본 확률표 |
