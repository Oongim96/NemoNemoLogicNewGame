# 05. 구역·칸 보상

[← 난이도 확장](./04-difficulty.md) · [목차](./README.md) · [다음: 잉크 덱 빌드 →](./06-ink-deck-build.md)

**정답 칸을 맞출 때마다** 그 칸의 보상이 **즉시** 연다. 구역 완료를 기다리지 않는다.

## 칸 보상 흐름

```txt
칸 선택 → 정답!
  → 보상 롤 (가중치)
  → [능력 3택1] | [골드] | [상점] | [이벤트] | [함정] | …
  → 퍼즐 계속 (덱·골드 반영됨)
```

**능력 3택1**이 가장 자주 나온다. 상세는 [잉크 덱 빌드](./06-ink-deck-build.md).

## 칸 보상 종류

| 보상 | 빈도(예) | 즉시 효과 |
| --- | --- | --- |
| **능력 3택1** | **~45%** | 잉크 카드 3장 중 1장 → 덱 추가 |
| 골드 | ~20% | 상점·덱 정리용 |
| 상점 | ~12% | 덱 구매·삭제·변환·강화 |
| 이벤트 | ~10% | 합성·교환·저주 등 |
| 회복 | ~5% | HP·액티브 충전 |
| 함정 | ~5% | 디버프, 저주 카드 |
| 보스 단서 | ~3% | 마지막 전투 약점·연계 |

칸 내용은 입장 전 **완전 공개되지 않음**. 캐릭터·덱 패시브로 힌트 가능.

## 구역 완료 보너스

구역 퍼즐 **전부** 완료 시:

- 큰 그림 해당 구역 **밝혀짐**
- 정확도·남은 턴에 따른 **보너스** (골드, 3택1 1회, 상점 할인 등)
- 구역 타입 공개 (칸 구성 성향 요약)

## 구역 타입 (칸 배치 성향)

| 타입 | 특징 |
| --- | --- |
| 보물 구역 | 3택1 등급 UP, 희귀 칸多 |
| 상점 구역 | 상점·골드 칸多 |
| 이벤트 구역 | 이벤트·합성 칸多 |
| 함정 구역 | 함정·저주 칸多 |
| 단서 구역 | 보스 단서 칸多 |
| 혼합 구역 | 랜덤 조합 |
