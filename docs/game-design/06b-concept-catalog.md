# 06b. 컨셉 카탈로그 (잉크 덱)

[← 잉크 덱 빌드](./06-ink-deck-build.md) · [목차](./README.md) · [자동 전투 →](./07-auto-battle.md)

각 **컨셉(태그)** 은 퍼즐·전투 모두 **완전히 다른 느낌**.  
같은 종류 카드는 규칙이 같고, **덱에 몇 장 모였는지** / **이번 턴에 몇 장 쐈는지** 에 따라 **공격 방식이 바뀐다.**

---

## 1. 턴 합산 — 카드 타입별 기본 규칙

| 타입 | 같은 카드·같은 컨셉이 겹칠 때 | 연출 느낌 |
| --- | --- | --- |
| **버프** | 수치 **합산** (+10% + 15% = +25%) | 오라가 겹쳐짐 |
| **공격** | **타격 횟수** +1 (피해는 카드별, 횟수만 늘음) | 연타·다발 |
| **실드** | 실드량 **합산** | 방패 겹침 |
| **스택** | 자원 **합산** (잉크, 불씨, 행운주사위…) | 게이지 충전 |
| **특수** | 카드마다 고유 — 아래 컨셉 참고 | 컨셉 고유 연출 |

한 턴에 `불꽃` 공격 3장 → **3연타** (각 타격 80이면 80×3, 버프 후 합산).  
한 턴에 `황금자` 버프 2장 → **+20% 합산**.

---

## 2. 이중 임계치 (덱 vs 턴)

모든 컨셉은 두 축을 가진다.

| 축 | 의미 | 예 |
| --- | --- | --- |
| **덱 임계** | 덱에 해당 컨셉 카드가 **총 몇 장** 있나 (쿨 무관) | `잉크` 덱 5장 → 퍼즐 중 최대 스택 +2 |
| **턴 임계** | **이번 턴** 쿨 0으로 **몇 장 발동**했나 | `잉크` 턴 3장 → 공격이 **잉크 범위**로 전환 |

덱 임계 = **런 전체 빌드 방향**, 턴 임계 = **그 턴 폭발력·연출 변화**.

---

## 3. 컨셉 8종 (핵심)

### 잉크 — 「퍼져 나가는 색」

**정체성:** 스택을 쌓고, 많을수록 **한 방이 넓어진다.**

| | |
| --- | --- |
| **퍼즐** | 칸 정답 시 인접 칸 힌트. 덱 `잉크` 3장+ → 십자 2칸 |
| **공격 기본** | 70 + 잉크 스택 1 |
| **버프** | 없음 (스택 자체가 자원) |

**턴 임계 (발동 장수)**

| 턴 발동 | 전투 변화 |
| --- | --- |
| 1~2장 | 단일 타겟 + 스택 |
| **3장+** | **잉크 범위**: 단일 피해의 60%가 옆 적(부위)에 튐 |
| **5장+** | **잉크 폭발**: 스택 전부 소모, 소모량×100 광역 (그 턴만) |

**덱 임계**

| 덱 장수 | 퍼즐·전투 패시브 |
| --- | --- |
| 3 | 잉크 최대 스택 +2 |
| 6 | 퍼즐: 줄 완성 시 잉크 +1 / 전투: 잉크 카드 쿨 -1 (최소 1) |

**대표 카드:** 연쇄 채색(CD1·공격+스택), 붉은 잉크(CD2·스택 소모 딜), 잉크 해일(CD3·턴3장+ 시 폭발 보조)

---

### 불꽃 — 「타격 수가 불길이 된다」

**정체성:** 공격 카드 = **연타**. 많이 모이면 **화상**으로 패턴이 바뀜.

| | |
| --- | --- |
| **퍼즐** | 연속 정답(콤보) 시 옆 줄 숫자 1개 오류 표시 |
| **공격** | 타격 1회 65. 같은 턴 불꽃 공격 N장 = **N연타** |
| **버프** | 없음 |

**턴 임계**

| 턴 연타 수 | 전투 변화 |
| --- | --- |
| 1~2 | 일반 연타 |
| **3~4** | 각 타격 **화상 1** 부여 (2턴 도트) |
| **5+** | **불길 통로**: 연타가 직선으로 이어지는 연출, 마지막 타격 ×2 |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 4 | 화상 최대 중첩 +2 |
| 7 | 퍼즐 콤보 5+ 시 전투 시작 화상 3 부여 |

**대표 카드:** 불씨 튀기(CD1·연타), 소각(CD2·연타+화상), 재의 선포(CD4·턴5연타+ 시 마지막×2 고정)

---

### 달빛 — 「보이지 않는 칸을 찌른다」

**정체성:** 적은 **약점**이 있고, 달빛은 **정밀·크리** 쪽. 연타보다 **한 방 각성**.

| | |
| --- | --- |
| **퍼즐** | 줄/열 완성 시 해당 줄 힌트 강조. 약점 칸 표시(희귀) |
| **공격** | 50, **크리 확률 +15%** (장당) |
| **버프** | 크리 피해 +20% (합산) |

**턴 임계**

| 턴 발동 | 전투 변화 |
| --- | --- |
| 1~2 | 크리 위주 |
| **3장+** | **월식**: 약점 부위 고정, 해당 부위 피해 +50% |
| **4장+ 버프 포함** | **만월**: 이번 턴 크리 100%, 연출 풀문 |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 3 | 전투 시작 약점 1개 공개 |
| 5 | 퍼즐: 구역당 약점 칸 1개 미리 표시 / 전투: 약점 파괴 시 실드 파쇄 |

**대표 카드:** 달빛 조각(CD1·소딜+크리), 월식의 눈(CD3·약점), 만월 기도(CD2·버프 합산)

---

### 철벽 — 「맞아도 덱이 산다」

**정체성:** 실드 **합산**. 많으면 **반격·무적 구간**.

| | |
| --- | --- |
| **퍼즐** | 실수 피해 감소. 실드형 버프 |
| **실드** | 35 (장당 합산) |
| **공격** | 40 (약함 — 탱커 컨셉) |

**턴 임계**

| 턴 실드+철벽 카드 | 전투 변화 |
| --- | --- |
| 실드 80+ | 이번 턴 받는 피해 **50% 감소** |
| **철벽 3장+ 발동** | **방벽 반격**: 적 공격 후 실드 30%만큼 되돌림 |
| **철벽 5장+ 덱** | 전투 첫 2턴 **무적** (덱 패시브, 1회성) |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 2 | 실드 합산 +15% |
| 4 | 적 반격 1회/전투 무효 |

**대표 카드:** 철벽 각인(CD2·실드), 거울 방패(CD3·반격), 요새의 심장(CD4·무적 패시브 조건)

---

### 행운 — 「주사위가 덱을 흔든다」

**정체성:** 분산·변수. **같은 카드도 턴마다 다른 숫자** — 대신 임계에서 **대박**.

| | |
| --- | --- |
| **퍼즐** | 3택1 재굴림, 보물 칸 확률 |
| **공격** | 40~120 **랜덤** (장당 1회) |
| **버프** | 다음 턴 랜덤 버프 1종 (합산 시 종류 추가) |

**턴 임계**

| 턴 발동 | 전투 변화 |
| --- | --- |
| 2장 | 각각 독립 랜덤 |
| **3장+** | **럭키 세븐**: 주사위 1번 더 굴려 **최고값**만 적용 |
| **4장+** | **잭팟**: 합산 피해가 보스 HP 15% 넘으면 골드 폭발 + 피해 ×1.5 |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 3 | 3택1에 희귀 1슬롯 |
| 6 | 퍼즐 칸 보상 골드 2배 확률 20% |

**대표 카드:** 황금 자(CD2·버프+골드), 주사위 던지기(CD1·랜덤 딜), 잭팟(CD3·턴4+ 연계)

---

### 힌트 — 「정보가 곧 화력」

**정체성:** 적 **방어력 깎기**·**미래 턴 예고**. 공격은 약하지만 **매 턴 누적 advantage**.

| | |
| --- | --- |
| **퍼즐** | 힌트 공개, 오답 무효, 숫자 분석 |
| **공격** | 45 + 적 방어 -5 (턴 중첩, 합산) |
| **버프** | 다음 턴 발동 카드 1장 쿨 -1 (희귀) |

**턴 임계**

| 턴 발동 | 전투 변화 |
| --- | --- |
| 2장 | 방어 깎기 합산 |
| **3장+** | **해독**: 적 버프 1개 제거 + 깎은 방어 2배 반영 |
| **덱5+ & 턴2+** | **예언**: UI에 **다음 턴 발동 카드** 전부 미리 표시 → 그 턴 피해 +30% |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 4 | 보스 단서 칸 확률 UP |
| 6 | 전투 시작 적 전체 스탯 -8% |

**대표 카드:** 깨진 돋보기(CD2·방어깎기), 예언의 필체(CD3·쿨예고), 무한 지우개(CD3·퍼즐)

---

### 격자 — 「줄 맞추면 전투도 맞춰진다」

**정체성:** 퍼즐 **줄/열 완성**이 전투 **배율**로 이어짐. 네모네모와 결속 가장 강한 컨셉.

| | |
| --- | --- |
| **퍼즐** | 줄 완성 보너스, 격자 강조 |
| **공격** | 55 + **런 내 줄 완성 수 × 3** (전투 시작 정산도 가능) |
| **버프** | 이번 턴 배율 +0.1 (합산) |

**턴 임계**

| 턴 발동 | 전투 변화 |
| --- | --- |
| 1~2 | 배율 적용 |
| **3장+** | **완성 포인트**: 퍼즐 줄 완성 1회당 이번 턴 +1 타격 |
| **5장+ 덱** | **거대 격자**: 피해가 보스 **격자 부위**별로 나뉘어 들어감 (약점 부위 2배) |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 3 | 줄 완성 시 골드 +2 |
| 5 | 구역 완료 시 전투 배율 영구 +0.05 (런 내) |

**대표 카드:** 줄 완성의 메아리(CD3), 격자 공명(CD2), 완성의 문양(CD4·거대 격자)

---

### 저주 — 「대가를 치르고 이긴다」

**정체성:** **자해·디버프** 대신 **폭력적 상한**. 하이리스크 빌드.

| | |
| --- | --- |
| **퍼즐** | 함정 칸 피해 → 저주 스택. 실수 시 스택+ |
| **공격** | 30 + **저주 스택 × 25** |
| **버프** | 공격 +40%, 받는 피해 +20% (합산 — 리스크도 합산) |

**턴 임계**

| 조건 | 전투 변화 |
| --- | --- |
| 저주 스택 3+ | 공격 **흡혈** 20% |
| **저주 카드 2장+ 턴 발동** | **역병**: 적에게 저주 이전, 스택 절반 이동 |
| **덱 4장+ & HP 50% 이하** | **필사**: 이번 턴 공격 2배, 다음 적 반격 2배 |

**덱 임계**

| 덱 장수 | 효과 |
| --- | --- |
| 2 | 저주 스택 상한 +3 |
| 5 | 함정 칸도 저주 1 스택 (리스크·리턴) |

**대표 카드:** 붉은 계약(CD2·공격+자해), 역병 인장(CD3), 필사의 각오(CD4)

---

## 4. 컨셉 교차 (2태그 카드)

한 카드에 태그 2개 → **두 컨셉 임계 모두** 카운트, 발동 시 **교차 보너스**.

| 조합 | 교차 보너스 (턴에 둘 다 발동 시) |
| --- | --- |
| `잉크` + `불꽃` | 연타마다 잉크 스택 +1, 마지막 타격에 스택 소모 |
| `달빛` + `힌트` | 약점 자동 표시 + 약점 피해 2배 |
| `철벽` + `격자` | 줄 완성 1당 실드 10 |
| `행운` + `저주` | 랜덤 피해 최소값이 저주 스택만큼 상승 |
| `잉크` + `격자` | 줄 완성 시 잉크 범위 1턴 early unlock |

---

## 5. 퍼즐에서도 컨셉이 다른 이유

같은 "힌트 1칸"이 아니라 **컨셉마다 퍼즐 체감이 다름**.

| 컨셉 | 퍼즐 체감 (한 줄) |
| --- | --- |
| 잉크 | 퍼져 나가며 칸 열림 |
| 불꽃 | 콤보·연속 정답 보상 |
| 달빛 | 약점·확정 정보 |
| 철벽 | 실수/forgiving |
| 행운 | 재굴림·도박 |
| 힌트 | 분석·숫자 읽기 |
| 격자 | 줄 완성 쾌감 극대화 |
| 저주 | 함정을 파워로 전환 |

---

## 6. 카드 설계 체크리스트

새 카드 추가 시:

- [ ] 컨셉 1~2개
- [ ] 타입: 공격 / 버프 / 실드 / 스택 / 특수
- [ ] 퍼즐 트리거 (이 컨셉만의 방식)
- [ ] 전투 쿨 1~4
- [ ] 턴 임계에 기여하는지 (연타? 합산? 스택?)
- [ ] 덱 3·5·7장 패시브와 충돌 없는지
