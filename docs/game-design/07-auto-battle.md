# 07. 자동 전투 시스템

[← 잉크 덱 빌드](./06-ink-deck-build.md) · [목차](./README.md) · [다음: 큰 그림 완성 →](./08-final-boss.md)

**한 플레이에 자동 전투는 1번.** 큰 그림 100% 완성 후.

퍼즐 = 손맛 + **잉크 덱** 쌓기, 전투 = 덱 + 파티 **뽕맛**.

---

## 1. 턴 구조 (핵심)

한 턴은 **「내 합산 공격 → 적 1회 반격」** 으로 고정. 플레이어 조작 없음.

```txt
[턴 N — 내 페이즈]
  ① 쿨 0인 잉크 카드 전부 수집
  ② 효과 합산·중첩 (공격 + 버프 + 스택 소모 등)
  ③ 한 번에 연출 → 적에게 피해·디버프 적용
  ④ 사용한 카드마다 개별 쿨타임 시작

[턴 N — 적 페이즈]
  ⑤ 보스/적 1회 공격 (또는 패턴 1스텝)

[턴 N+1]
  쿨이 돌아온 카드만 다시 ①~⑤ 반복
```

**매 턴 "쓸 수 있는 카드를 전부 쓴다"** — 다만 **쿨이 안 돌아온 카드는 빠진다.**  
그래서 턴마다 쓰이는 카드 조합이 달라지고, 덱을 짤 때 **쿨 분포**가 빌드의 일부가 된다.

### 왜 이 구조인가

| 장점 | 설명 |
| --- | --- |
| **뽕맛** | 한 턴에 여러 카드가 **합쳐져** 큰 숫자·화려한 연출 |
| **가독성** | 턴당 내 공격 1번 + 적 공격 1번 — 정리됨 |
| **덱 빌딩** | 쿨 1 짧은 카드 vs 쿨 3 강한 카드 **리듬**이 생김 |
| **턴 변주** | T1 만발 → T2 쿨로 비는 카드多 → T3 다시 풀세트 |

---

## 2. 카드 합산·중첩 규칙

쿨 0인 카드를 **동시에** 전부 발동 → **컨셉별로 다르게** 합친 뒤 한 턴의 공격으로 연출.

→ 컨셉별 상세: **[컨셉 카탈로그](./06b-concept-catalog.md)**

### 타입별 기본 (공통)

| 타입 | 같은 턴 여러 장 |
| --- | --- |
| **버프** | 수치 **합산** (+10% + 15% = +25%) |
| **공격** | **타격 횟수** +1 (피해는 타격마다 계산 후 합산) |
| **실드** | 실드량 **합산** |
| **스택** | 잉크·저주·화상 등 자원 **합산** |

### 컨셉 임계 (공격 **방식**이 바뀜)

| 컨셉 | 예: 이번 턴 많이 쐈을 때 |
| --- | --- |
| `잉크` 3장+ | 단일 → **범위** (튐) |
| `불꽃` 연타 5+ | **불길 통로**, 마지막 ×2 |
| `달빛` 3장+ | **월식** 약점 집중 |
| `철벽` 3장+ | **반격** |
| `격자` 3장+ | 줄 완성 수만큼 **추가 타격** |

**덱에 몇 장 모였는지**는 패시브·상한·퍼즐 보정 (컨셉 카탈로그 「덱 임계」).

### 발동 순서 (내부)

```txt
1. 덱·턴 임계로 **공격 방식** 결정 (범위/연타/약점…)
2. 버프·디버프 합산
3. 스택 생성/소모
4. 타격 횟수만큼 피해 계산 → 합산
5. 실드 적용
6. 최종 피해 → 적
7. 사용 카드 쿨 시작
```

## 3. 카드 쿨타임

| 쿨 (턴) | 성향 | 예시 |
| --- | --- | --- |
| **0~1** | 매 턴 거의 참여 | 「연쇄 채색」소형 딜, 잉크+1 |
| **2** | 격턴마다 | 「붉은 잉크」중형 딜 |
| **3** | 한 번 쓰고 쉼 | 「줄 완성의 메아리」대형 딜, 파티 실드 |
| **4+** | ult급 | 에픽 카드, 보스 단서 연계기 |

- **같은 카드 2장** in 덱 → **각각 독립 쿨** (T1에 2장 다 쓰면 T2~T3 둘 다 쿨)
- **파티 ult**: 쿨 무시 1회 발동 or 전체 쿨 -1 등 (캐릭터별)
- 카드 **강화**: 피해 UP or **쿨 -1** (상한 1)

```txt
덱: [연쇄채색 CD1] [연쇄채색 CD1] [붉은잉크 CD2] [철벽 CD3]

T1: 세 장 전부 → 합산 520 + 실드  →  쿨 시작
T2: 연쇄×2만   → 합산 160          →  붉은잉크·철벽은 쿨 중
T3: 연쇄×2 + 붉은잉크 → 합산 600   →  철벽만 쿨 중
T4: 전부 다시  → 풀 버스트
```

---

## 4. 파티 역할

| 요소 | 턴 내 위치 |
| --- | --- |
| **잉크 덱** | 쿨 0 카드 전부 합산 공격의 주체 |
| **파티 기본** | 매 턴 소량 고정 딜 (쿨 없음) or 덱 합산에 가산 |
| **파티 ult** | 조건 충족 시 해당 턴 합산에 **추가** (연출 강조) |
| **태그 시너지** | 같은 턴 동시 발동 태그 수에 따라 배율 |

## 5. 연출·UI

```txt
┌─ 턴 3 — 내 공격 ─────────────────────────────┐
│  발동: 불씨×3  소각×1  황금자×1                 │
│  불꽃 연타 4 → 【화상】 / 버프 합산 +10%         │
│  타격 65×4 = 260 → 286                          │
│  합산 ► 286 (4연타)  ──►  [고대 용]             │
└───────────────────────────────────────────────┘
        ↓
┌─ 적 반격 ────────────────────────────────────┐
│  용의 숨결  -82  (실드 흡수 30)                 │
└───────────────────────────────────────────────┘
```

- **합산 숫자**를 크게 — 뽕맛의 중심
- 발동 카드 아이콘 나열 + **다음 턴 예고**(쿨 1 남은 카드 표시)

## 6. 전투 예시 (전체)

```txt
[큰 그림 100%] 고대 용 완성
[자동 전투] 덱 12장 | 선제: 실드 50

T1  [연쇄×2 + 붉은잉크 + 황금자버프] → 680  /  용 반격 -45
T2  [연쇄×2 + 줄완성메아리]          → 520  /  용 반격 -60
T3  [연쇄×2 + 붉은잉크 + 루나ult]    → 1420 /  용 반격 -40
T4  [전 카드 풀세트 + 잉크폭발시너지]  → 2100 /  용 반격 -55
…
[승리]
```

---

## 7. 밸런스·주의점

| 이슈 | 대응 |
| --- | --- |
| 쿨 1만 모으면 단조로움 | 쿨 2~3 고화력 카드·시너지가 **턴마다 조합이 바뀌게** |
| 턴이 너무 김 | 보스 HP·카드 합산량으로 **4~8턴** 목표 |
| 버프 영구 중첩 | 버프 카드도 **쿨** + 지속은 N턴 별도 |
| 덱 18장 전부 T1 발동 | 쿨 다양화 + 덱上限 + 합산 diminishing (선택) |
