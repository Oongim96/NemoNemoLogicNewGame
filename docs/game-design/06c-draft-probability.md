# 06c. 카드 드래프트 확률 (3택1)

[← 잉크 덱 빌드](./06-ink-deck-build.md) · [목차](./README.md) · [카드 목록](../../data/cards/ink-cards.md)

**드래프트** = **구역 완료 시** 카드 **3장 제시 → 1장 선택**.

확률은 **2단계**로 나뉜다.

```txt
[1단계] 구역 클리어 보상이 뭔지?  →  드래프트 ~72% / 골드 / 상점 / 이벤트 가끔 …
[2단계] 드래프트면 어떤 카드 3장?  →  아래 공식 (파티·컨셉·고유카드)
```

---

## 1단계: 구역 클리어 보상 종류

| 보상 | 확률(기본) | 비고 |
| --- | --- | --- |
| **카드 드래프트** | **72%** | 3장 중 1장 선택 + 골드 10~18 |
| 골드만 | 6% | 추가 골드 12~25 |
| 상점 | 10% | |
| 이벤트 | 5% | 가끔 |
| 회복 | 4% | |
| 함정 | 3% | |

구역 타입·캐릭터 스킬로 위 비율은 소폭 변동 가능.  
**2단계는 드래프트가 떴을 때만** 적용.

---

## 2단계: 드래프트 3장 뽑기 (핵심)

### 한 줄 요약

> **모든 컨셉 카드는 나올 수 있지만**, 편성 파티 **주 컨셉은 가중치 UP**, **파티 캐릭 고유 카드는 초반에 거의 보장에 가깝게**.

### 후보 풀 (Pool)

드래프트 1회마다:

1. `enabled = 1` 인 카드만
2. **덱에 이미 상한** (`max_per_deck`) 찬 카드 제외
3. **등급 상한** — 런 진행에 따라 (아래 표)
4. `start_00x` 시작 덱 전용 카드 제외 (`draft_weight = 0`)

| 런 진행 (구역 클리어 수) | 드래프트에 나올 수 있는 등급 |
| --- | --- |
| 0 ~ 2 | 일반 위주 (희귀 **최대 1슬롯**) |
| 3 ~ 6 | 일반 + 희귀 |
| 7 ~ 12 | 일반 + 희귀 + 에픽 (에픽 **낮은 가중**) |
| 13+ | 전 등급 |

### 카드 1장당 가중치 공식

```
최종 가중치 = 기본 가중치
            × 컨셉 배율 (파티)
            × 고유 카드 배율 (초반 부스트)
            × 신규 컨셉 배율 (덱 다양성, 선택)
            × 이미 덱에 많은 카드 감쇠 (선택)
```

#### ① 기본 가중치

카드 마스터의 `draft_weight` (일반 85~120, 희귀 50~70, 에픽 25~30).

#### ② 컨셉 배율 (파티 연동)

파티 4인 각각 **주 컨셉 태그** 1~2개를 합산한다.

| 카드와 파티 관계 | 배율 | 예 |
| --- | --- | --- |
| 카드 **주 컨셉** = 파티에 **2명 이상** 해당 | **×2.2** | 세라핀+루나 둘 다 `잉크` 계열 파티… |
| 카드 **주 컨셉** = 파티 **1명** 해당 | **×1.8** | 파티에 `달빛` 루나 1명 |
| 카드 **부 컨셉**만 파티와 일치 | **×1.3** | |
| 파티와 **무관**한 컨셉 | **×0.85** | 나오긴 하지만 살짝 희귀 |

→ **전 컨셉 출현 가능**, 주 컨셉이 **눈에 띄게 자주**.

**파티 컨셉 표 (기본)**

| 캐릭터 | 주 컨셉 | 부 컨셉 |
| --- | --- | --- |
| 루나 | 달빛 | 힌트 |
| 브릭스 | 철벽 | — |
| 미오 | 행운 | — |
| 세라핀 | 잉크 | — |

#### ③ 고유 카드 배율 (가챠 · 초반 거의 확보)

캐릭터마다 **고유 잉크 카드** 1~2장 (가챠 해금 → 드래프트 풀 등록).

| 조건 | 배율 |
| --- | --- |
| **아직 덱에 없는** 파티원 고유 카드 & 구역 0~1 | **×8** |
| 같은 & 구역 2~4 | **×4** |
| 같은 & 구역 5~8 | **×2** |
| 같은 & 구역 9+ | **×1.2** |
| 이미 덱에 있음 (중복 허용 카드) | **×0.4** |
| **파티에 없는** 캐릭터 고유 카드 | **×0** (이번 런 풀 제외) |
| 해금 안 한 고유 카드 | **×0** (메타 미해금) |

**초반 거의 준다** = 구역 2~3 안에 **각 파티원 고유 1장씩** 나올 확률이 매우 높음.

##### 고유 카드 매핑 (예시)

| 캐릭터 | 고유 카드 ID | 카드명 |
| --- | --- | --- |
| 루나 | moon_004 | 월식의 눈 |
| 루나 | moon_005 | 만월 기도 |
| 브릭스 | wall_004 | 거울 방패 |
| 미오 | luck_005 | 럭키 세븐 |
| 세라핀 | ink_006 | 잉크 해일 |

(추가 고유 카드는 가챠 캐릭터마다 `character_unique_cards` 데이터로 확장)

##### 하드 가드 (선택 적용 권장)

| 시점 | 규칙 |
| --- | --- |
| **런 최초 드래프트** | 3장 중 **1슬롯** = 파티 고유 카드 전용 풀에서 강제 추첨 (미보유 우선) |
| **구역 3 클리어까지** | 파티 고유 **미보유**가 1장이라도 남으면, 드래프트마다 **최소 1장** 고유 후보에 **×8** 이상 적용 (위 표) |

→ “초반에 편성 캐릭 카드 거의 다 모은다” 체감.

#### ④ 신규 컨셉 · 중복 감쇠 (유기적 느낌, 선택)

| 상황 | 배율 |
| --- | --- |
| 현재 덱에 **없는** 컨셉의 카드 | **×1.15** |
| 같은 `card_id` 이미 1장 보유 | **×0.5** (2장째는 의도적 빌드만) |

## 3장 뽑는 순서 (중복 없음)

```txt
1. 후보 풀 + 최종 가중치 계산
2. 가중치 랜덤으로 1장 → 슬롯 A (풀에서 제거)
3. 남은 풀에서 2장 → 슬롯 B
4. 남은 풀에서 3장 → 슬롯 C
5. 슬롯 규칙 보정 (아래)
```

### 슬롯 규칙

| 규칙 | 내용 |
| --- | --- |
| **다양성** | 3장 전부 같은 `card_id` 불가 (이미 제거로 해결) |
| **희귀 슬롯** | 구역 0~2: 3장 중 **최대 1장**만 희귀 이상 (미오 패시브 시 **2번째 슬롯** 희귀 허용) |
| **파티 컨셉 1장 보장** (부드러운 규칙) | 3장 모두 파티 무관 컨셉이면 → **가장 가중치 높은 1장**을 파티 주 컨셉 카드로 **교체** (1회) |

## 계산 예시

**파티:** 루나(달빛·힌트) + 세라핀(잉크) + 브릭스(철벽) + 미오(행운)  
**진행:** 구역 1 (초반)  
**드래프트 발동**

| 카드 | 기본 | 컨셉 | 고유 | 합산 배율 | 비고 |
| --- | --- | --- | --- | --- | --- |
| ink_001 연쇄 채색 | 120 | ×1.8 잉크 | ×1 | 216 | 세라핀 |
| moon_004 월식의 눈 | 55 | ×1.8 달빛 | **×8** 루나 미보유 | **792** | **압도** |
| flame_001 불씨 튀기 | 115 | ×0.85 | ×1 | 98 | off-theme |
| wall_001 철벽 각인 | 110 | ×1.8 | ×1 | 198 | 브릭스 |
| luck_001 주사위 | 115 | ×1.8 | ×1 | 207 | 미오 |

→ 이 풀에서 3장 뽑으면 **월식의 눈**이 거의 항상 포함.  
나머지 2장은 잉크·철벽·행운·off-theme이 **경쟁** → 유기적이면서 파티 색깔 유지.

**구역 10+** 이후: 고유 ×1.2로 떨어져 **일반 빌드 카드·에픽** 비중이 올라감.

## 의사코드

```txt
function openDraft(party, run):
  pool = filterEnabledCards()
  pool -= cardsAtMaxCopy(run.deck)
  pool = applyGradeCap(pool, run.sectionsCleared)
  pool -= notUnlockedByMeta(player)

  for card in pool:
    w = card.draft_weight
    w *= partyConceptMult(card, party)      // 0.85 ~ 2.2
    w *= uniqueEarlyMult(card, party, run)  // 0 ~ 8
    w *= varietyMult(card, run.deck)         // optional
    card.weight = w

  slots = []
  if run.isFirstDraft:
    slots.add( pickFrom(partyUniquesNotOwned, party) )

  while slots.size < 3:
    slots.add( weightedPickWithoutReplacement(pool) )

  slots = applySlotRules(slots, party, run)  // rare cap, concept soft guarantee
  return slots
```

## 메타·가챠 연동

| 메타 | 드래프트 영향 |
| --- | --- |
| 캐릭터 해금 | 해당 캐릭 **고유 카드** 풀 등록 |
| 고유 카드 영구 해금 | `draft_weight` 기본값 +10 |
| 캐릭터 중복 성장 | 주 컨셉 배율 +0.1 / 드래프트 재굴림 +1 |
| 미오 | 희귀 슬롯 +1, 행운 컨셉 +0.2 배율 |
