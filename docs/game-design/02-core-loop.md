# 02. 핵심 게임 루프

[← 게임 개요](./01-overview.md) · [목차](./README.md) · [다음: 거대 그림 던전 →](./03-dungeon-system.md)

## 한 플레이의 구조

```txt
[입장] 파티 편성 + 시작 잉크 덱
   ↓
[큰 그림 1장] — 구역별 퍼즐
   ↓
[구역 퍼즐 1개] 풀기 → 완료 시 보상 1회 (3택1·상점·이벤트…) → 잉크 덱 성장
   ↓              └ 큰 그림 일부 밝힘
[반복]
   ↓
[큰 그림 100%] → 완성 연출
   ↓
[자동 전투 1회] — 잉크 덱 + 파티 자동 발동 (뽕맛)
   ↓
[클리어 보상]
```

## 구역 퍼즐 중 (전투 없음)

1. 구역 선택 → **네모네모 로직 1개** 풀기
2. 구역 **100% 완료** → 보상 롤 1회 — 주로 **능력 3택1** → [잉크 덱](./06-ink-deck-build.md)에 추가
3. 상점 보상 → 덱 **구매·삭제·변환·강화**
4. 이벤트 보상 → 합성·교환·저주
5. 큰 그림 해당 구역 **밝혀짐** + 클리어 골드

## 큰 그림 완성 후

6. 완성 연출 → **자동 전투 1회** (턴마다 쿨 0 카드 **합산** → 적 1타)

## 연결 요약

| | 퍼즐 중 | 마지막 전투 |
| --- | --- | --- |
| 잉크 덱 | 패시브·줄 발동 (추후) | 카드 자동 연쇄 |
| 상점 | 덱 편집 | — |
| 파티 | 스킬·3택1 보정 | ult·태그 증폭 |
