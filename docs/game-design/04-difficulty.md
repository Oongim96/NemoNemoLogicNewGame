# 04. 난이도 확장 구조

[← 거대 그림 던전](./03-dungeon-system.md) · [목차](./README.md) · [다음: 구역·칸 보상 →](./05-sections-and-rewards.md)

난이도는 **두 축**으로 점진적으로 올라간다.  
튜토리얼 없이 **작은 네모네모 로직 하나**부터 시작.

## 축 1: 구역 퍼즐 크기 (먼저)

**큰 그림의 한 구역** = 네모네모 로직 퍼즐 1개.  
처음엔 작은 퍼즐로 규칙을 익히고, **구역당 격자 크기**만 키운다.

| 단계 | 구역 퍼즐 크기 | 특징 |
| --- | --- | --- |
| 시작 | 3x3 ~ 5x5 | 기본 칠하기·X·힌트 |
| 성장 | 6x6 ~ 8x8 | 캐릭터 스킬, 유물, 칸 보상 |
| 중반 | 10x10 ~ 12x12 | 제한 턴, 실수 피해 |
| 후반 | 13x13 ~ 15x15 | 함정 칸, 힌트 왜곡 |

## 축 2: 큰 그림의 구역 수 (이후)

구역 퍼즐에 익숙해지면, **큰 그림을 몇 칸으로 나눌지** 키운다.  
2x2 → 3x3 → … → **엔드 10x10 구역**.

| 큰 그림 구역 | 구역 개수 | 특징 |
| --- | --- | --- |
| 2x2 | 4 | 짧은 런, 큰 그림 맛보기 |
| 3x3 | 9 | 칸 보상·상점·이벤트 본격 |
| 4x4 | 16 | 빌드 다양성, 구역 선택 전략 |
| 5x5 ~ 7x7 | 25 ~ 49 | 단서·함정 구역 |
| 8x8 ~ 10x10 | 64 ~ 100 | **엔드**, 마지막 전투 **뽕맛** |

구역 수가 늘수록 구역 퍼즐 크기도 함께 성장 (예: 10x10 구역 맵 → 각 구역 13~15칸)

## 진행 예시

```txt
[1] 5x5 퍼즐 1구역짜리 큰 그림 → 밝히기 → (완성 시) 자동 전투 1회
[2] 2x2 구역 큰 그림, 각 구역 5x5~8x8 → 4구역 밝히기 → 자동 전투 1회
[3] 3x3 구역, 각 8x8~10x10 → 9구역 밝히기 → 자동 전투 1회
   …
[N] 10x10 구역, 각 13x13~15x15 → 100구역 밝히기 → 대형 연출 → 자동 전투 1회
```

**매 단계마다 자동 전투는 딱 1번** — 큰 그림이 100% 완성된 뒤.

## 난이도 상승 요소

- 구역 퍼즐 격자 크기
- 큰 그림 구역 수 (더 많은 칸 보상·빌드 기회)
- 제한 턴·실수 피해·힌트 왜곡
- 잠긴 구역, 클리어 순서 조건
- 마지막 자동 전투 난이도 (보스 스탯·패턴)
