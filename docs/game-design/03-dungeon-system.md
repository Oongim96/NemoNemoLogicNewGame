# 03. 거대 그림 던전 시스템

[← 핵심 게임 루프](./02-core-loop.md) · [목차](./README.md) · [다음: 난이도 확장 →](./04-difficulty.md)

## 기본 구조: 큰 그림 하나

**한 플레이 = 네모네모 로직 큰 그림 1장.**

던전이 여러 개의 독립된 맵이 아니라, **하나의 거대한 그림**이다.  
이 그림은 **구역** 격자로 나뉘며, 각 구역마다 별도의 네모네모 로직 퍼즐이 붙어 있다.

**예시:** 4x4 구역, 전체 테마 "고대 용"

```txt
큰 그림 맵 (밝혀진 / 어두운 구역)
┌────┬────┬────┬────┐
│ ░░ │ ░░ │ ██ │ ██ │   ░░ = 아직 안 푼 구역
├────┼────┼────┼────┤   ██ = 퍼즐 완료, 밝혀짐
│ ░░ │ ░░ │ ██ │ ░░ │
├────┼────┼────┼────┤
│ ██ │ ░░ │ ░░ │ ░░ │
├────┼────┼────┼────┤
│ ██ │ ░░ │ ░░ │ ░░ │   ← 마지막 구역까지 채우면
└────┴────┴────┴────┘      큰 그림 완성
```

플레이어는 처음에 그림 전체가 뭔지 모른다.  
구역을 하나씩 밝힐수록 **큰 그림의 실루엣과 정체**가 드러나고, 완성될 보스·테마를 추리하게 된다.

## 구역 = 독립 퍼즐

- 큰 그림의 **한 칸(구역)** = 플레이어가 푸는 **네모네모 로직 퍼즐 1개**
- 구역 퍼즐 크기는 **현재 난이도**에 맞게 생성 (3x3 ~ 15x15)
- 구역 완료 → 큰 그림에서 해당 부분 **밝혀짐** (조각 맞추기 연출)
- **전투는 구역마다 없음** — 퍼즐과 보상만

## 구역 완료 = 보상

구역 퍼즐을 **전부 완료**하면 보상이 **1회** 열린다.

- 보상 롤 → 3택1·상점·이벤트·골드 등 → [잉크 덱](./06-ink-deck-build.md)
- 동시에 큰 그림 해당 구역 **밝혀짐**

## 큰 그림 완성

모든 구역을 밝히면 **큰 그림 100%**.

- 조각들이 이어지며 **하나의 완성 이미지** (고대 용, 별자리, 저주받은 성 …)
- 그림이 **살아 움직이는** 대형 완성 연출
- 이어서 **자동 전투 1회** (이 게임에서 유일한 전투)
