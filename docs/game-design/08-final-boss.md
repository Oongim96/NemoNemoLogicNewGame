# 08. 큰 그림 완성과 자동 전투

[← 자동 전투](./07-auto-battle.md) · [목차](./README.md) · [다음: 캐릭터·가챠 →](./09-characters-gacha.md)

마지막 구역 완료 → 큰 그림 100% → **덱 쇼케이스** 자동 전투.

## 완성 순간

1. 마지막 구역 퍼즐 클리어
2. 전 구역 **대형 완성 연출** — 그림이 살아 움직임
3. **자동 전투 1회** — 잉크 덱 + 파티 ult **뽕맛**

## 테마별 예시

- **고대 용:** 단서 카드 → 약점 UI → 잉크 폭발 연계
- **별자리:** `달빛` 태그 덱 → 별령 소환 패턴
- **저주받은 성:** 저주 카드多 빌드 → 고리스크 고데미지
