## 한 줄 소개

큰 그림을 **구역마다 퍼즐 1개**로 밝히며 **잉크 덱**을 쌓고, 구역 클리어 시 **3택1·상점·이벤트**로 빌드한 뒤 **자동 전투 1회**로 마무리하는 로그라이크 RPG.

## 요약

```txt
구역 퍼즐 완료 → 3택1/상점/이벤트 (맵에 미리 표시) → 잉크 덱 성장
반복 → 큰 그림 밝히기
전 구역 완료 → 자동 전투 1회 (덱 + 파티)
```

## 차별화

- **잉크 덱** — 퍼즐·전투 **동일 카드**, 덱빌딩 로그라이크
- **구역 클리어 3택1** — 로그라이크 빌드의 주 재료
- **상점 = 덱 편집** — 구매·삭제
- **8컨셉** — 덱·턴 임계로 전투 방식 변화
