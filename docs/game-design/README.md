## 요약

```txt
칸 정답 → 3택1/상점/이벤트 (즉시) → 잉크 덱 성장
구역 완료 → 큰 그림 밝히기
전 구역 완료 → 자동 전투 1회 (덱 + 파티 뽕맛)
```

> 칸을 맞추면 카드가 쌓이고, 덱이 퍼즐을 돕고, 그림이 완성되면 덱이 싸운다.

## 차별화

- **잉크 덱** — 퍼즐·전투 **동일 카드**, 덱빌딩 로그라이크
- **칸마다 3택1** — 네모네모보다 재밌는 런 빌드
- **상점 = 덱 편집** — 구매·삭제·변환·강화
- **8컨셉** — 버프 합산 / 공격 연타 / 덱·턴 임계로 **방식 변화**
