# 09. 캐릭터와 가챠 시스템

[← 큰 그림 완성](./08-final-boss.md) · [목차](./README.md) · [다음: 로그라이크 →](./10-roguelike.md)

캐릭터 = **퍼즐 패시브** + **전투 패시브** + **컨셉 증폭** + (한정) **스피드·덱 시너지**.

---

## 1. 캐릭터가 하는 일

| 역할 | 설명 |
| --- | --- |
| **퍼즐 패시브** | 네모네모를 **빠르·안전**하게 — 힌트, 자동 해석, 실수 완화 |
| **전투 패시브** | 잉크 덱 **컨셉 증폭** — 태그 배율, 임계치 완화 |
| **고유 카드** | 드래프트·시작 덱에 들어가는 **전용 잉크 카드** |
| **ult** | 구역당 1~2회 등 — 위기 탈출·대형 퍼즐 보조 |

난이도가 오르면 **풀 구역·큰 격자** → 플레이타임 길어짐.  
캐릭터 + 덱 = **스피드 메타** (빨리 넘기기) / 한정 캐릭 = 그 정점.

---

## 2. 등급

| 등급 | 역할 | 획득 |
| --- | --- | --- |
| **R** | 연습·이벤트 (추후) | 기본 지급 |
| **SR** | **기본 로스터** 4인 | 스토리·기본 가챠 |
| **SSR 한정** | **컨셉별 1인** (8명) | 한정 가챠 · 강한 패시브 |
| **SSR 일반** | (추후 확장) | 통합 풀 |

**한정 SSR** = 능력 **아주 좋음**. 퍼즐 스피드 + 덱 시너지 둘 다.

→ 상세: [한정 캐릭터 8인](../../data/characters/limited-characters.md)

## 3. 기본 로스터 (SR)

무료·초반에 쓰는 **4인**. 한정보다 약하지만 컨셉 체험 가능.

| ID | 이름 | 컨셉 | 퍼즐 패시브 | 전투 패시브 |
| --- | --- | --- | --- | --- |
| char_luna | 루나 | 달빛·힌트 | 줄 1개 힌트 자동 분석 | `달빛` 카드 재발동 |
| char_brix | 브릭스 | 철벽 | 실수 피해 -50% | `철벽` 실드 +20% |
| char_mio | 미오 | 행운 | 드래프트 1회 재굴림/런 | `행운` 크리 +15% |
| char_sera | 세라핀 | 잉크 | 완성률에 따라 HP 회복 | `잉크` 스택당 힐 |

→ [기본 로스터 상세](../../data/characters/base-roster.md)

| 컨셉 | 캐릭터 | 한 줄 |
| --- | --- | --- |
| 잉크 | **베가** | 칸 맞출 때마다 번짐 — 빠른 채색 |
| 불꽃 | **이그니스** | 콤보로 줄 폭발 |
| 달빛 | **디아나** | 약점 줄 통째로 읽기 |
| 철벽 | **마그너** | 실수 무시하고 밀기 |
| 행운 | **포춘** | 4택1·드래프트 지배 |
| 힌트 | **오라클** | 숫자 자동 해석 |
| 격자 | **그리드** | 줄 완성 연쇄 |
| 저주 | **모이라** | 함정을 파워로 |

## 5. 파티 × 덱 × 드래프트

```
편성 4인 → 주 컨셉 가중치 (드래프트)
         → 패시브 중첩 (퍼즐·전투)
         → 고유 카드 초반 등장

런 중 드래프트 → 잉크 덱 성장
              → 캐릭터 패시브가 같은 컨셉 카드 증폭
```

| 예시 파티 | 스피드·빌드 방향 |
| --- | --- |
| 베가 + 세라핀 + 잉크 덱 | 잉크 범위·폭발 초반 완성 |
| 오라클 + 그리드 + 격자·힌트 덱 | 대형 퍼즐 **줄 단위** 속공 |
| 마그너 + 브릭스 + 철벽 덱 | 실수 없이 빠르게 밀기 |

## 6. 가챠

| 풀 | 내용 |
| --- | --- |
| **일반** | SR 캐릭터, 잉크 카드, 스킨 |
| **한정** | 해당 컨셉 **SSR 1인** + 그 컨셉 카드 rate up |
| **중복** | 한정 SSR: 패시브 강화 / 고유 카드 ++ / 스킨 |

**한정 캐릭 해금 시**

- 고유 잉크 카드 1~2장 → 드래프트 풀 영구 등록
- 메타 패시브 일부 영구 (예: 해당 컨셉 드래프트 +5%)

---

## 7. 긴 플레이 · 이어하기

10x10급·대형 격자는 **기존 네모네모처럼 오래** 걸릴 수 있음.

| 기능 | 내용 |
| --- | --- |
| **이어하기** | 런 중단 시 — 큰 그림 진행, 덱, HP, **풀던 구역 퍼즐 상태** 저장 |
| **구역 단위 저장** | 구역 완료마다 자동 체크포인트 |
| **퍼즐 중 저장** | 칸·X·스택·남은 ult 등 그대로 복원 |
| **스피드 설계** | 한정+덱으로 **한 구역 시간 단축** → 맵 전체는 길어도 **구역당 부담↓** |

난이도 ↑ = 구역 수 ↑ → **이어하기 필수**에 가깝게 설계.

---

## 8. 스피드 메타 (의도)

| 레이어 | 스피드 기여 |
| --- | --- |
| 한정 SSR 퍼즐 패시브 | 자동 해석·번짐·줄 연쇄 |
| SR 기본 | 실수 완화·소규모 힌트 |
| 잉크 덱 | 칸 보상·컨셉 효과로 퍼즐 단축 |
| ult | 막힌 구역 1회 돌파 |

**목표:** 숙련+한정+덱이 맞으면 **같은 격자도 체감 30~50% 단축** (튜닝 값).
