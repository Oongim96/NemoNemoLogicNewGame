# 컨셉 임계치

[← 카드 목록](./ink-cards.md) · [데이터 README](./README.md)

덱에 모인 장수 / 이번 턴 발동 장수에 따라 패시브·공격 방식이 바뀐다.

---

## 잉크

| 조건 | 효과 |
| --- | --- |
| 덱 3장+ | 잉크 최대 스택 +2 |
| 덱 6장+ | 줄 완성 시 잉크 +1 / 잉크 카드 쿨 -1 (최소 1) |
| **턴 3장+** | 단일 피해 60% 스플래시 |
| **턴 5장+** | 잉크 스택 전부 소모 ×100 광역 |

## 불꽃

| 조건 | 효과 |
| --- | --- |
| 덱 4장+ | 화상 최대 중첩 +2 |
| 덱 7장+ | 퍼즐 콤보 5+ 시 전투 시작 화상 3 |
| **턴 연타 3~4** | 타격마다 화상 1 |
| **턴 연타 5+** | 마지막 타격 ×2 (불길 통로) |

## 달빛

| 조건 | 효과 |
| --- | --- |
| 덱 3장+ | 전투 시작 약점 1개 공개 |
| 덱 5장+ | 구역당 약점 칸 표시 / 약점 시 실드 파쇄 |
| **턴 3장+** | 월식 — 약점 부위 피해 +50% |
| **턴 4장+ (버프 포함)** | 만월 — 이번 턴 크리 100% |

## 철벽

| 조건 | 효과 |
| --- | --- |
| 덱 2장+ | 실드 합산 +15% |
| 덱 4장+ | 적 반격 1회 무효 |
| **턴 실드 80+** | 받는 피해 50% 감소 |
| **턴 3장+** | 실드 30% 반격 |
| 덱 5장+ | 첫 2턴 무적 (1회) |

## 행운

| 조건 | 효과 |
| --- | --- |
| 덱 3장+ | 3택1 희귀 슬롯 +1 |
| 덱 6장+ | 칸 골드 2배 확률 20% |
| **턴 3장+** | 럭키 세븐 — 랜덤 2회 최고값 |
| **턴 4장+** | 잭팟 — 피해 15% HP 넘으면 ×1.5 + 골드 |

## 변동

| 조건 | 효과 |
| --- | --- |
| 덱 3장+ | 모든 랜덤 **최솟값 +5** |
| 덱 5장+ | 랜덤 **최댓값 +15%** |
| **턴 3장+** | 낮은 랜덤 1회 재굴림 |
| **턴 4장+** | 이번 턴 랜덤 최소 = (최소+최대)/2 |

## 힌트

| 조건 | 효과 |
| --- | --- |
| 덱 4장+ | 보스 단서 칸 확률 UP |
| 덱 6장+ | 전투 시작 적 스탯 -8% |
| **턴 3장+** | 해독 — 적 버프 제거 + 방어깎기 2배 |
| 덱 5+ & 턴 2+ | 예언 — 다음 턴 발동 카드 예고 + 피해 +30% |

## 격자

| 조건 | 효과 |
| --- | --- |
| 덱 3장+ | 줄 완성 골드 +2 |
| 덱 5장+ | 구역 완료마다 전투 배율 +0.05 (런 내) |
| **턴 3장+** | 줄 완성 1회당 추가 타격 +1 |
| 덱 5장+ | 거대 격자 — 부위별 피해 / 약점 2배 |

## 저주

| 조건 | 효과 |
| --- | --- |
| 덱 2장+ | 저주 스택 상한 +3 |
| 덱 5장+ | 함정 칸도 저주 +1 |
| **저주 스택 3+** | 공격 흡혈 20% |
| **턴 2장+** | 역병 — 저주 절반 적 이전 |
| 덱 4+ & HP 50%↓ | 필사 — 공격 2배 / 다음 반격 2배 |

## 교차 보너스 (2태그 카드)

| 조합 | 턴에 둘 다 발동 시 |
| --- | --- |
| 잉크 + 불꽃 | 연타마다 잉크 +1, 마지막 타격에 스택 소모 |
| 달빛 + 힌트 | 약점 자동 표시 + 약점 피해 2배 |
| 철벽 + 격자 | 줄 완성 1당 실드 10 |
| 행운 + 저주 | 랜덤 최소값 = 저주 스택 |
| 잉크 + 격자 | 줄 완성 시 잉크 범위 1턴 early unlock |
