## 3택1 가중치 참고

`draft_weight` = **기본 가중치**만 담음. 실제 드래프트 확률은 **파티 컨셉·고유 카드·진행도**로 곱해짐.

→ **[드래프트 확률 설계](../../docs/game-design/06c-draft-probability.md)** 참고

| draft_weight | 의미 (기본값) |
| --- | --- |
| 100~120 | 자주 등장 (일반 다수) |
| 50~70 | 희귀 |
| 25~30 | 에픽 |
| 0 | 시작 덱 전용 |

같은 카드 덱 상한: 대부분 **2장**, 에픽·일부 희귀 **1장**.
