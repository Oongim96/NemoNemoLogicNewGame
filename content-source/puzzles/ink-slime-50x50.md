# 잉크 슬라임 — 50×50 예시 퍼즐

## 크기 공식

```
완성 그림 = mapSize × puzzleSize
         = 5 × 10 = 50  (한 변 기준)
```

| 항목 | 값 | 의미 |
| --- | --- | --- |
| **구역당 퍼즐** | 10×10 | 맵에서 한 칸을 클릭하면 풀 네모네모 로직 |
| **맵 (총 구역)** | 5×5 | 25개 구역 |
| **완성 그림** | 50×50 | 25조각을 이으면 되는 최종 픽셀 아트 |

> 잘못된 예: 2×2 맵 + 3×3 퍼즐 → 완성 **6×6** (9×9 아님)

---

## 데이터

| 파일 | 역할 |
| --- | --- |
| [ink-slime-50x50.json](./ink-slime-50x50.json) | `mapSize: 5`, `puzzleSize: 10`, `generateMaster: slime_blob` |
| `npm run sync-data` | 50×50 masterGrid 생성 → 25개 10×10 `sections` |
| `slime_50x50` | 런타임 퍼즐 세트 ID |

---

## 스테이지

| 스테이지 | mapSize | puzzleSize | 완성 |
| --- | --- | --- | --- |
| 잉크 슬라임 (예시) | 5 | 10 | **50×50** |
| 슬라임 숲 (튜토리얼) | 1 | 10 | **10×10** (왼쪽 위 한 조각) |
| 고대 용의 그림 | 2 | 3 | **6×6** |
| 용의 전설 | 3 | 3 | **9×9** |

---

## 플레이

1. **그림 선택** → 「잉크 슬라임 (예시)」
2. **5×5 맵** — 25구역, 각 10×10 퍼즐
3. 전부 클리어 → **50×50** 슬라임 완성

`masterGrid`를 직접 넣으려면 JSON에 `generateMaster` 대신 50×50 `0`/`1` 배열을 작성합니다.
