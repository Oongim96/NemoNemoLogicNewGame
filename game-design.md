# 네모네모 로직 로그라이크 RPG 기획서

## 1. 게임 개요

### 가칭

**픽셀 던전: 잉크 오브 운명**

### 한 줄 소개

네모네모 로직 퍼즐을 풀어 던전 지도를 밝히고, 각 방의 보상을 획득하며, 완성된 거대한 그림을 다시 해석해 최종 보스를 클리어하는 로그라이크 RPG 가챠 게임.

### 핵심 컨셉

이 게임에서 던전은 단순한 방들의 집합이 아니라, 하나의 거대한 네모네모 로직 그림이다.  
플레이어는 큰 그림을 여러 조각으로 나눈 방을 하나씩 공략하고, 각 방의 퍼즐을 완성해 전체 그림을 복원한다.

전체 그림이 완성되면, 그 그림 자체가 최종 기믹이 되어 마지막 풀이와 보스전으로 이어진다.

---

## 2. 핵심 게임 루프

1. 던전에 입장한다.
2. 전체 맵은 하나의 큰 네모네모 로직 그림으로 구성된다.
3. 큰 그림은 여러 개의 방 조각으로 나뉘어 있다.
4. 플레이어는 원하는 방을 선택해 해당 구역의 퍼즐을 푼다.
5. 방을 클리어하면 조각이 전체 그림에 합쳐지고, 랜덤 보상을 획득한다.
6. 전투, 상점, 이벤트, 보물, 함정 등 방의 정체가 공개된다.
7. 모든 방을 클리어하면 전체 그림이 완성된다.
8. 완성된 그림을 다시 해석하는 최종 퍼즐 또는 보스전이 열린다.
9. 클리어 후 보상과 성장 재화를 얻고, 다음 난이도 또는 다음 던전에 도전한다.

---

## 3. 거대 그림 던전 시스템

### 기본 구조

던전 전체는 하나의 큰 그림이며, 각 방은 그림의 일부다.

예시: 전체 그림이 "고대 용의 문장"인 던전

```txt
┌────┬────┬────┬────┐
│ 날개│ 날개│ 머리│ 뿔  │
├────┼────┼────┼────┤
│ 몸통│ 몸통│ 눈  │ 입  │
├────┼────┼────┼────┤
│ 꼬리│ 발톱│ 보물│ 문  │
├────┼────┼────┼────┤
│ 비늘│ 비늘│ 심장│ 보스│
└────┴────┴────┴────┘
```

플레이어는 처음부터 그림의 정체를 알 수 없다.  
방을 하나씩 클리어할수록 전체 그림의 일부가 드러나고, 플레이어는 완성될 그림과 최종 기믹을 추리하게 된다.

### 방 조각

각 방은 작은 네모네모 로직 퍼즐이다.

- 방 퍼즐을 풀면 해당 조각이 완성된다.
- 완성된 조각은 전체 그림에 합쳐진다.
- 조각마다 랜덤 보상과 이벤트가 배정된다.
- 일부 방은 전체 그림의 중요한 단서가 된다.

### 전체 그림 완성

모든 조각을 완성하면 큰 그림이 공개된다.

예시:

- 고대 용
- 봉인된 왕관
- 거대한 문장
- 저주받은 성
- 잊힌 신의 얼굴
- 별자리 지도

완성된 그림은 단순한 결과 화면이 아니라, 최종 단계의 퍼즐 힌트가 된다.

---

## 4. 난이도 확장 구조

난이도가 올라갈수록 전체 맵 크기와 방 퍼즐 크기가 함께 커진다.

| 진행 단계 | 전체 맵 크기 | 방 퍼즐 크기 | 특징 |
| --- | --- | --- | --- |
| 초반 | 3x3 방 | 5x5 | 기본 규칙 학습 |
| 중반 | 4x4 방 | 10x10 | 방 타입과 랜덤 보상 확대 |
| 후반 | 5x5 방 | 15x15 | 함정, 엘리트, 특수 규칙 등장 |
| 엔드 콘텐츠 | 6x6 이상 | 15x15 이상 | 보스 기믹, 왜곡 힌트, 제한 턴 추가 |

### 난이도 상승 요소

- 전체 그림의 크기 증가
- 방 개수 증가
- 방 하나당 퍼즐 크기 증가
- 제한 턴 추가
- 실수 피해 증가
- 힌트 왜곡
- 잠긴 방 등장
- 특정 순서로 풀어야 하는 방 등장
- 보스 방 해금 조건 추가

---

## 5. 방 타입과 랜덤 보상

각 방은 입장 전까지 완전히 공개되지 않는다.  
플레이어는 위험도, 예상 보상, 주변 방 정보 등을 보고 선택한다.

### 방 타입

- 전투 방
- 엘리트 전투 방
- 보스 단서 방
- 상점 방
- 이벤트 방
- 보물 방
- 함정 방
- 회복 방
- 강화 방
- 미스터리 방

### 보상 예시

- 골드
- 장비 조각
- 캐릭터 성장 재료
- 임시 유물
- 퍼즐 힌트권
- 체력 회복
- 상점 할인권
- 보스 약점 단서
- 가챠 재화

### 랜덤 보상 구조

각 방은 다음 요소를 조합해 보상을 결정한다.

- 방 타입
- 현재 난이도
- 퍼즐 풀이 정확도
- 남은 체력
- 사용한 스킬 수
- 현재 보유 유물
- 캐릭터 특성

예를 들어, 보물 방을 실수 없이 클리어하면 추가 보상이 붙고, 함정 방에서 실수하면 체력 피해와 함께 보상이 감소한다.

## 6. 퍼즐 배틀 시스템

전투는 턴제 RPG처럼 진행되지만, 공격과 방어는 네모네모 로직 풀이와 연결된다.

### 기본 규칙

- 정확히 칠한 칸은 공격력으로 전환된다.
- 연속 정답은 콤보가 된다.
- 실수한 칸은 반격 피해를 발생시킨다.
- 특정 모양을 완성하면 스킬이 발동된다.
- 제한 턴 안에 완성하면 추가 보상을 얻는다.

### 전투 예시

몬스터가 등장하면 10x10 또는 15x15 퍼즐이 나타난다.

```txt
      1  3  2  1
    ┌──────────┐
  2 │ ■ ■ □ □ □│
  1 │ □ □ ■ □ □│
  3 │ ■ ■ ■ □ □│
  1 │ □ □ □ ■ □│
  2 │ □ ■ □ ■ □│
    └──────────┘
```

퍼즐을 잘 풀수록 적에게 더 큰 피해를 주고, 실수할수록 적의 반격이 강해진다.

## 7. 전체 그림 최종 풀이

모든 방을 클리어해 전체 그림이 완성되면 최종 단계가 열린다.

### 최종 단계 예시

#### 완성 그림: 고대 용

- 용의 눈 부분을 다시 선택해 약점을 찾는다.
- 비늘 패턴의 순서를 해석해 봉인을 해제한다.
- 날개, 꼬리, 발톱 조각의 위치가 보스 공격 패턴 힌트가 된다.
- 최종 보스가 완성된 그림의 형태로 등장한다.

#### 완성 그림: 별자리 지도

- 별자리 선을 올바른 순서로 연결해야 한다.
- 잘못 연결하면 추가 전투가 발생한다.
- 정확히 연결하면 숨겨진 보스 약점이 공개된다.

#### 완성 그림: 저주받은 성

- 창문, 문, 첨탑의 위치를 이용해 탈출 경로를 찾는다.
- 특정 방 조각에서 얻은 단서를 조합해야 한다.
- 최종 보스 방의 퍼즐 힌트가 전체 성 그림 안에 숨어 있다.

### 핵심 재미

작은 퍼즐을 푸는 행위가 단기 목표가 되고, 전체 그림 완성이 중기 목표가 되며, 완성된 그림을 다시 해석하는 과정이 최종 목표가 된다.

## 8. 캐릭터와 가챠 시스템

캐릭터는 단순한 능력치 차이가 아니라, 퍼즐을 푸는 방식 자체를 바꾼다.

### 캐릭터 예시

#### 루나, 달빛 해커

- 클래스: 마법사
- 기본 능력: 한 줄의 힌트를 자동 분석한다.
- 궁극기: 3x3 영역의 정답 여부를 표시한다.

#### 브릭스, 철벽 기사

- 클래스: 탱커
- 기본 능력: 퍼즐 실수 피해를 감소시킨다.
- 궁극기: 틀린 칸 하나를 자동 수정한다.

#### 미오, 행운의 고양이

- 클래스: 도적
- 기본 능력: 보물 방 출현 확률을 증가시킨다.
- 궁극기: 방 보상을 한 번 재굴림한다.

#### 세라핀, 잉크 성녀

- 클래스: 힐러
- 기본 능력: 퍼즐 완성률에 따라 체력을 회복한다.
- 궁극기: 실수 횟수를 초기화한다.

### 가챠 방향

가챠는 캐릭터, 스킨, 펫, 퍼즐 이펙트, 장비 조각 등을 제공한다.

권장 방향:

- 캐릭터마다 고유한 퍼즐 조작 능력을 가진다.
- 저등급 캐릭터도 특정 던전이나 빌드에서 강력하게 활용된다.
- 중복 캐릭터는 단순 능력치보다 선택지를 넓히는 방향으로 사용한다.

중복 성장 예시:

- 스킬 사용 후 공개되는 칸의 방향 선택
- 보상 재굴림 횟수 증가
- 특정 방 타입에서 추가 효과 발동
- 보스전 전용 특성 개방

## 9. 로그라이크 요소

매번 던전 구조, 방 타입, 보상, 퍼즐 규칙이 달라진다.

### 랜덤 요소

- 전체 그림 테마
- 방 배치
- 방 보상
- 몬스터 속성
- 임시 유물
- 이벤트 선택지
- 상점 상품
- 보스 패턴

### 유물 예시

#### 깨진 돋보기

던전 시작 시 무작위 칸 3개를 공개한다.

#### 붉은 잉크병

실수할 때마다 공격력이 증가하지만 체력이 감소한다.

#### 무한 지우개

전투당 한 번 오답을 무효화한다.

#### 황금 자

한 줄을 완성할 때마다 골드를 획득한다.

#### 뒤집힌 지도

방 보상이 더 좋아지지만, 일부 방 타입이 입장 전까지 숨겨진다.

## 12. 차별화 포인트

- 던전 전체가 하나의 네모네모 로직 그림이다.
- 각 방은 큰 그림의 일부이자 독립된 퍼즐이다.
- 방마다 랜덤 보상과 이벤트가 있어 로그라이크성이 강하다.
- 퍼즐 풀이가 전투, 탐험, 보상 획득과 직접 연결된다.
- 완성된 그림을 다시 해석해 최종 보스를 클리어한다.
- 캐릭터 스킬이 퍼즐 규칙과 풀이 방식을 바꾼다.
- 가챠 캐릭터가 단순 전투력이 아니라 전략적 플레이 스타일을 제공한다.

## 13. 요약

이 게임은 "작은 퍼즐을 풀어 큰 그림을 완성하고, 완성된 큰 그림을 다시 해석해 최종 클리어하는" 구조를 가진다.

네모네모 로직의 추리 재미, 로그라이크의 랜덤성, RPG의 성장감, 가챠의 수집 재미를 하나의 시스템 안에 엮는 것이 핵심이다.

> 퍼즐을 풀면 방이 열리고, 방을 모으면 그림이 되고, 그림을 해석하면 운명이 열린다.
